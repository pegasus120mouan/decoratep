# Dossier Images DECORATEP

Ce dossier contient les images du site web DECORATEP.

## Fichiers requis :
- `logo.jpg` - Logo principal de DECORATEP (à placer ici)
- `favicon.ico` - Icône du site (optionnel)

## Instructions :
1. Placez le fichier `logo.jpg` fourni dans ce dossier
2. Le logo sera automatiquement utilisé dans :
   - Header de navigation
   - Footer du site
   - Hero section de la page d'accueil

## Dimensions recommandées :
- Logo : 300x100px minimum (format horizontal recommandé)
- Favicon : 32x32px ou 16x16px
